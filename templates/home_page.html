Main View

The Main view provides a user-friendly interface for quickly logging completed exams and displays feedback regarding the amount of work completed. It includes a date-picker, shift-picker, and tools for efficient logging.

Management View

The Management view allows users to manage procedures, folders, and shifts. Users can create, edit, and delete procedures, organize them into folders, and define shifts for better organization.

Reports View

In the Reports view, users can generate detailed reports based on the logged procedures. The view includes a sortable table, date range filters, and tools to filter data based on shifts, modalities, descriptions, and CPT codes.
